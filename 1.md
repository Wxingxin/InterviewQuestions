下面给你一份 **JavaScript `typeof` 运算符的使用大全**，包括语法、所有返回值、使用场景、注意事项、常见坑点与最佳实践。

---

# 📌 一、`typeof` 基本语法

```js
typeof operand
```

或更常见形式：

```js
typeof(operand)
```

两者完全等价。

---

# 📌 二、`typeof` 的所有可能返回值（共 8 种）

| 返回值           | 说明            | 示例                         |
| ------------- | ------------- | -------------------------- |
| `"number"`    | 数字（包括 NaN）    | `typeof 1`                 |
| `"string"`    | 字符串           | `typeof "abc"`             |
| `"boolean"`   | 布尔值           | `typeof true`              |
| `"undefined"` | 未定义           | `typeof undefined`         |
| `"object"`    | 对象类型（含 null！） | `typeof {}`, `typeof null` |
| `"function"`  | 函数            | `typeof function(){}`      |
| `"symbol"`    | ES6 Symbol    | `typeof Symbol()`          |
| `"bigint"`    | 大整数类型         | `typeof 123n`              |

# 📌 四、`typeof` 的优点

| 特性         | 描述                                           |
| ---------- | -------------------------------------------- |
| 安全判断未声明变量  | `typeof x === "undefined"` 不报 ReferenceError |
| 简单、轻量      | 基础类型判断很方便                                    |
| 可判断函数类型    | `typeof fn === 'function'`                   |
| 支持 ES6+ 类型 | bigint, symbol                               |

# 📌 六、`typeof` 使用场景（最佳实践）

## ⭐ 1. 判断基础类型

```js
if (typeof name === "string") { ... }
```

## ⭐ 2. 判断函数（最常用）

```js
if (typeof callback === "function") { ... }
```

## ⭐ 3. 判断变量是否声明过（不会报错）

```js
if (typeof window.foo === "undefined") {
    // safe
}
```

## ⭐ 4. 模块判断（如 Node/浏览器）

```js
if (typeof window !== "undefined") {
    // 运行在浏览器
}
```

## ⭐ 5. 判断 BigInt / Symbol

```js
if (typeof id === "bigint") { ... }
```

# 📌 七、结合示例的 `typeof` 速查表（最全）

```js
typeof 1                 // "number"
typeof NaN               // "number"
typeof 'abc'             // "string"
typeof true              // "boolean"
typeof undefined         // "undefined"
typeof Symbol()          // "symbol"
typeof 123n              // "bigint"

typeof {}                // "object"
typeof []                // "object"
typeof null              // "object"
typeof new Date()        // "object"
typeof /re/g             // "object"
typeof new Map()         // "object"

typeof function(){}      // "function"
typeof class A{}         // "function"
typeof () => {}          // "function"

typeof x                 // "undefined"（x 未声明）
```

# 📌 八、补充：判断所有类型的更强方式（相比 typeof 更可靠）

如果你需要判断“精确类型”，推荐用：

```js
Object.prototype.toString.call(value)
```

也可以封装成：

```js
const type = v => Object.prototype.toString.call(v).slice(8, -1);
```
